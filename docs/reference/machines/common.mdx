<h4 id="settings.networking.vpn.cidr">
  <span class="hidden">`settings.networking.vpn.cidr`</span>
</h4>

<ResponseField
    name="settings.networking.vpn.cidr"
    type="string"
    default={"10.100.0.0/24"}
    
    >
CIDR that defines the VPN network.

It is also required to determine the machine IP address from the machine ID on the VPN.

For instance, if the CIDR is `10.100.0.0/24` and `settings.vpn.id` is `5`, then the machine IP address will be `10.100.0.5`.


Declared in [modules/networking/vpn/common.nix](https://github.com/plmercereau/nicos/blob/main/modules/networking/vpn/common.nix#L34).
